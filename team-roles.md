# Team Roles

> Roles are flexible. People can cover more than one area, and advisors can contribute part-time. Meeting the official RMPC eligibility requirements is required for anyone joining the application.

## Organizer / Project Lead

**Dan Lee-Odinson** — project management, proposal coordination, documentation, budget tracking, timeline, requirements tracking, team coordination, and submission workflow.

## Technical contributor roles

| Role | Focus | Typical contribution |
| --- | --- | --- |
| Mechanical / aerospace engineer | Structure, interfaces, mechanisms | Payload geometry, grapple/mate features, load paths |
| Electrical / embedded systems | Power, data, microcontrollers | Power/data validation, sensing, telemetry |
| Robotics / controls | Manipulation planning | Task sequencing, success criteria, arm interface logic |
| CAD / mechanical design | Modeling and drawings | 3D models, drawings, design-for-manufacture |
| Fabrication (CNC / welding / rapid prototyping) | Making parts | Prototype hardware, fixtures, mockups |
| Space systems / payload integration | System-level fit | Interface compliance, integration planning |
| Thermal / power / mass budgeting | Resource budgets | Mass/power/thermal tracking against limits |
| Environmental test | Qualification | Vibration, thermal/vacuum, materials guidance |
| Payload operations / CONOPS | Mission operations | Concept of operations, flight-test planning |
| Flight hardware / test planning | Verification | Test plans, acceptance criteria |

## Advisory roles

- Faculty advisor
- Industry mentor
- Subject-matter reviewer (any of the above areas)

## How contribution is recognized

- Public credit in the repository and application (where appropriate and permitted).
- A documented, portfolio-quality engineering project regardless of selection outcome.
- Any prize-related compensation would be governed by a **written team agreement** established before submission or award acceptance. This is volunteer work with no guaranteed funding.
