# Requirements Notes

> A running checklist of things to pull from the official RMPC materials and confirm before drafting the application. **Do not treat any number here as authoritative** — every item must be verified against the official sources.

## Sources to read first

- Official RMPC site: https://rmpc.nasatechleap.org/
- Technical guidelines: https://rmpc.nasatechleap.org/technical-guidelines/
- Evaluation criteria: https://rmpc.nasatechleap.org/evaluation-criteria/
- Rules, terms, and conditions: https://rmpc.nasatechleap.org/rules-terms-and-conditions/
- Insurance guidelines: https://rmpc.nasatechleap.org/insurance-guidelines/

## Key dates (confirm against official site)

- Registration deadline: **July 29, 2026**
- Phase 1 application deadline: **August 12, 2026**
- Award pathway: up to three winners, up to **$500,000** across phases
- Flight-test pathway: NASA intends to provide hosted orbital flight tests for winners

## Requirements to extract and track

### Eligibility
- [ ] Who is eligible (individuals, teams, students, organizations)?
- [ ] Citizenship / residency constraints, if any
- [ ] Team size or composition limits
- [ ] Registration vs. application requirements

### Technical / interface
- [ ] Host robotic arm and grapple/interface standard
- [ ] Mass envelope
- [ ] Volume / dimensional envelope
- [ ] Power budget and interface
- [ ] Data interface and telemetry expectations
- [ ] Thermal limits
- [ ] Materials / outgassing restrictions
- [ ] Safety and hazard requirements

### Verification / test
- [ ] Vibration qualification requirements
- [ ] Thermal/vacuum requirements
- [ ] Acceptance vs. qualification testing expectations
- [ ] Any required documentation or data deliverables

### Evaluation
- [ ] Scoring categories and weights
- [ ] What reviewers prioritize (feasibility, innovation, ISAM relevance, etc.)
- [ ] Required application sections and page/format limits

### Insurance / financial responsibility
- [ ] Insurance or demonstrated financial responsibility requirements for winners
- [ ] When these must be resolved (application vs. award stage)

## Budget items to start estimating

- Prototype materials and fabrication
- Any required test services (vibration, thermal/vacuum)
- Software / tooling
- Documentation and submission costs
- Contingency
